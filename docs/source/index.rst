
data_structure_visualizer
=====================================

Version: |release|

A Custom Jupyter Widget Library


Quickstart
----------

To get started with data_structure_visualizer, install with pip::

    pip install data_structure_visualizer

or with conda::

    conda install data_structure_visualizer
